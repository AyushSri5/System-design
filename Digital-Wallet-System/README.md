# Improvements

## Design Patterns

- Factory Pattern
  - Can be used for different types of payment methods like credit card,bank accounts etc.
- Singleton Pattern
  - Can be used for ensuring single instance of critical components like account and transactions system.
- Adapter Pattern
  - Can be used for integrating multiple payment gateways like credit card,online payment etc.
- Observer Pattern
  - Can be used for notifying on events like failed transaction , deposit funds and withdraw funds.
- Chain of Responsibility Pattern
  - Can be used for transaction processing where multiple things like fraud detection,authorization and fund transfer.
